# 好友系统

好友系统允许玩家添加好友、私聊和组队。

## 好友规则

### 添加好友

1. 发送好友申请
2. 对方接受申请
3. 双向确认成为好友

### 好友限制

| 限制类型 | 值 |
|----------|------|
| 最大好友数 | 50 |
| 申请有效期 | 24 小时 |
| 拉黑后 | 无法发送消息 |

## 好友功能

### 私聊
- 通过 QQ 或 WebSocket 发送私信
- 消息实时推送

### 组队
- 邀请好友组队
- 组队探索秘境
- 组队对战

### 好友状态
- 在线状态显示
- 最后上线时间
- 游戏内状态（修炼中、战斗中、游历中）

## QQ 指令

| 指令 | 说明 |
|------|------|
| `/好友` | 查看好友列表 |
| `/好友 添加 <玩家名>` | 发送好友申请 |
| `/好友 接受 <玩家名>` | 接受好友申请 |
| `/好友 删除 <玩家名>` | 删除好友 |
| `/好友 私聊 <玩家名> <消息>` | 发送私聊消息 |

### REST API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/game/friend/list` | 查看好友列表 |
| POST | `/api/game/friend/request` | 发送好友申请 |
| POST | `/api/game/friend/accept` | 接受好友申请 |
| DELETE | `/api/game/friend/{playerName}` | 删除好友 |
