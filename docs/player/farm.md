# 农场系统

农场系统是玩家获取资源的重要途径。玩家拥有自己的农田，可以种植各类种子，通过浇水施肥提升产量。

## 农田规则

| 参数 | 值 |
|------|------|
| 初始地块数 | 4 块 |
| 最大地块数 | 12 块 |
| 水分上限 | 100%（每秒消耗 1%） |
| 肥力上限 | 100%（消耗强化石提升） |
| 扩建费用 | 100 × 2^(当前地块数-4) 金币 |

### 扩建费用计算

```
扩建费用 = 100 × 2^(当前地块数 - 4)
```

| 当前地块数 | 扩建费用 |
|------------|----------|
| 4 → 5 | 100 金币 |
| 5 → 6 | 200 金币 |
| 6 → 7 | 400 金币 |
| 7 → 8 | 800 金币 |
| 8 → 9 | 1600 金币 |
| 9 → 10 | 3200 金币 |
| 10 → 11 | 6400 金币 |
| 11 → 12 | 12800 金币 |

## 种植流程

1. **选择地块**：选择一个空闲地块
2. **选择种子**：从背包中选择要种植的种子
3. **开始种植**：消耗种子，地块进入生长状态
4. **浇水施肥**：保持水分和肥力在较高水平
5. **等待成熟**：根据种子类型等待生长时间
6. **收获作物**：成熟后收获作物，获得奖励

## 产量计算

```
实际产量 = 基础产量 × (水分系数 + 肥力系数) × 幸运值
```

| 因素 | 系数范围 |
|------|----------|
| 水分系数 | 0.5（0%水分）~ 1.5（100%水分） |
| 肥力系数 | 0.5（0%肥力）~ 2.0（100%肥力） |
| 幸运值 | 0.8 ~ 1.2（随机） |

### 示例计算

种植千年人参：
- 基础产量：2-3
- 水分系数（80%水分）：1.3
- 肥力系数（60%肥力）：1.3
- 幸运值：1.1

实际产量 = 2.5 × (1.3 + 1.3) × 1.1 ≈ 7.15 → 7 个

## 农场操作

### QQ 指令

| 指令 | 说明 |
|------|------|
| `/种田` | 查看农场状态 |
| `/种田 种植 <地块> <种子>` | 在指定地块种植种子 |
| `/种田 浇水 <地块>` | 给指定地块浇水（+30%水分） |
| `/种田 施肥 <地块>` | 给指定地块施肥（消耗强化石，+20%肥力） |
| `/种田 收获 <地块>` | 收获指定地块的作物 |
| `/种田 清理 <地块>` | 清理枯萎的作物 |
| `/种田 扩建` | 扩建农田（消耗金币） |

### REST API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/game/farm/plots` | 查看农田状态 |
| POST | `/api/game/farm/plant` | 种植种子 |
| POST | `/api/game/farm/water` | 浇水 |
| POST | `/api/game/farm/fertilize` | 施肥 |
| POST | `/api/game/farm/harvest` | 收获 |
| POST | `/api/game/farm/clear` | 清理 |
| POST | `/api/game/farm/expand` | 扩建 |

## 种植策略

### 新手策略
- 优先种植灵草种子（生长快，产量稳定）
- 及时浇水，保持水分在 60% 以上
- 逐步扩建农田，积累资源

### 进阶策略
- 种植高价值作物（幽冥花、天山雪莲）
- 保持肥力在 80% 以上，最大化产量
- 合理安排种植时间，避免作物枯萎

### 效率最大化
- 使用闹钟提醒收获时间
- 利用离线时间种植长周期作物
- 优先种植当前需求的材料
