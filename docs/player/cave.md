# 洞府系统

洞府是修仙者的修炼圣地，可提供修炼加成、灵气收集和额外存储空间。

## 开辟洞府

玩家达到筑基期后，可花费300灵石开辟自己的洞府。

```
/洞府 create [名称]
```

**要求**:
- 境界达到筑基期（第3境界）
- 消耗300灵石
- 每个玩家只能拥有一个洞府

## 洞府等级

## 灵气系统

洞府会自动汇聚天地灵气，每10秒增加10点灵气（受等级上限限制）。

### 收集灵气

```
/洞府 collect
```

每5分钟可收集一次，将洞府中的灵气转化为灵石。

**冷却时间**: 5分钟

## 洞中冥想

在洞府中打坐冥想可获得经验值，享受修炼加成。

```
/洞府 meditate
```

**经验公式**: `100 + (修炼加成 × 10)`

## 洞府功能总览

| 功能 | 说明 |
|------|------|
| 开辟洞府 | 筑基期+300灵石 |
| 等级升级 | 提升各项属性，消耗灵石 |
| 修炼加成 | 每级+5%经验加成 |
| 存储扩展 | 每级+10格背包容量 |
| 灵气收集 | 自动汇聚，定期收集转化为灵石 |

## 与背包系统联动

洞府的存储加成会叠加到背包容量中：

```
总容量 = 基础30格 + (等级-1)×5格 + 空间戒指(+20格) + 洞府存储加成
```

例如：10级洞府 + 空间戒指 = 额外增加120格容量
