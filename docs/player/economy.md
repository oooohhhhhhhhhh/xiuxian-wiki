# 经济系统

经济系统包含金币、灵石两种货币，以及灵庄（银行）存储功能。

## 货币类型

### 金币

| 属性 | 值 |
|------|------|
| 用途 | 基础消费（农田扩建、装备强化） |
| 获取方式 | 游历、种田、任务奖励 |
| 存储 | 可存入灵庄 |

### 灵石

| 属性 | 值 |
|------|------|
| 用途 | 高级消费（坊市交易、宗门创建、重铸灵根） |
| 获取方式 | 秘境探索、PVP奖励、出售物品 |
| 存储 | 直接存储于角色 |

## 灵庄系统

灵庄是玩家存储金币的安全场所，支持存入、取出和查询功能。

### 存入金币

```
/灵庄 存入 <金额>
```

| 参数 | 说明 |
|------|------|
| 金额 | 要存入的金币数量 |

**存入规则**：
- 每次存入会生成唯一的存款编号
- 存款编号用于后续取出操作
- 存款无利息

### 取出金币

```
/灵庄 取出 <存款编号> <金额>
```

| 参数 | 说明 |
|------|------|
| 存款编号 | 存入时系统返回的编号 |
| 金额 | 要取出的金币数量 |

**取出规则**：
- 必须使用正确的存款编号
- 取出金额不能超过存款余额
- 取出后存款记录更新余额

### 查询存款

```
/灵庄 查询
```

查看所有存款记录，包括：
- 存款编号
- 存款金额
- 当前余额

## QQ 指令

| 指令 | 说明 |
|------|------|
| `/灵庄 存入 <金额>` | 存入金币到灵庄 |
| `/灵庄 取出 <编号> <金额>` | 从灵庄取出金币 |
| `/灵庄 查询` | 查看灵庄存款信息 |

## 经济策略

### 新手期
- 优先积累金币扩建农田
- 适当存储金币到灵庄

### 中期
- 积极参与坊市交易获取灵石
- 利用土灵根减少手续费

### 后期
- 重铸灵根提升属性
- 参与宗门战获取丰厚奖励
