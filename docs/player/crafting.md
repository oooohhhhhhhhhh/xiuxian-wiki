# 制造系统

制造系统包括炼丹和炼器，玩家可以通过配方合成各种物品。

## 制造规则

### 成功率计算

```
成功率 = 基础成功率 + 熟练度加成 + 材料品质加成
失败安慰 = 获得少量制造经验（基础经验的 1/3）
```

| 因素 | 加成效果 |
|------|----------|
| 基础成功率 | 根据配方（15% ~ 95%） |
| 熟练度加成 | 每级 +2%（最高 +20%） |
| 材料品质加成 | 使用高品质材料 +5% |

### 熟练度

- 使用制造技能获得熟练度
- 熟练度达到一定值升级
- 等级越高，成功率越高

### 品质系统

制造成功时有概率获得高品质物品：

| 品质 | 概率 | 效果加成 |
|------|------|----------|
| 普通 | 默认 | 基础效果 |
| 高品质 | 根据配方 | +20% ~ +50% 效果 |
| 最高品质 | 根据配方 | +20% ~ +50% 效果 |

### 灵石换算

制造消耗的灵石会自动从高阶灵石换算：

| 灵石等级 | 价值换算 |
|----------|----------|
| 低阶灵石 | 1 基础单位 |
| 中阶灵石 | 100 低阶灵石 |
| 高阶灵石 | 10000 低阶灵石 |
| 至尊灵石 | 1000000 低阶灵石 |

## 制造策略

### 新手策略

1. **优先制造回血丹和回蓝丹** - 基础消耗品，需求大
2. **积累制造经验** - 通过低级配方提升熟练度
3. **逐步尝试高级配方** - 境界提升后解锁更多配方

### 进阶策略

1. **批量制造高需求物品** - 强化石、传送符等消耗品
2. **利用熟练度加成** - 高熟练度大幅提升成功率
3. **关注市场需求** - 制造畅销物品获取利润

### 高级策略

1. **挑战高难度配方** - 仙丹、龙剑等传说物品
2. **使用保护符** - 高价值物品制造时减少损失
3. **搭配称号** - 丹道尊者等称号提升制造效率

## 制造注意事项

1. **境界要求** - 每个配方需要达到对应境界才能使用
2. **材料消耗** - 制造失败材料仍然会消耗
3. **经验获取** - 即使失败也能获得少量经验
4. **品质影响** - 高品质物品属性更强
5. **灵石换算** - 系统自动从高阶灵石换算，无需手动兑换
