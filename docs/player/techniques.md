# 功法系统

功法系统是修仙世界的核心系统之一，通过学习和修炼功法，玩家可以获得永久属性加成和特殊效果。

## 功法类型

| 类型 | 说明 | 主要加成 |
|------|------|----------|
| ATTACK | 攻击型 | 攻击力、战斗伤害加成 |
| DEFENSE | 防御型 | 防御力、生命值、伤害减免 |
| CULTIVATION | 修炼型 | 修炼速度、经验加成 |
| UTILITY | 辅助型 | 灵力、特殊效果 |

## 功法升级

### 升级费用

功法升级消耗金币和灵石，费用随等级递增：

| 等级 | 金币消耗 | 灵石消耗 |
|------|----------|----------|
| 1 → 2 | 基础金币 × 1 | 基础灵石 × 1 |
| 2 → 3 | 基础金币 × 2 | 基础灵石 × 2 |
| 3 → 4 | 基础金币 × 3 | 基础灵石 × 3 |
| ... | ... | ... |
| n → n+1 | 基础金币 × n | 基础灵石 × n |

**升级公式**: `升级费用 = 基础费用 × 当前等级`

### 熟练度系统

使用功法可以获得熟练度，熟练度达到一定值后可以升级：

| 等级 | 熟练度需求 | 效果加成 |
|------|------------|----------|
| 1 | 0 | 基础效果 |
| 2 | 100 | 效果 +10% |
| 3 | 200 | 效果 +20% |
| ... | ... | ... |
| 10 | 900 | 效果 +100% |

**熟练度获取**:
- 修炼时自动获得熟练度
- 战斗中使用技能获得熟练度
- 使用功法秘籍获得熟练度

## 功法学习

### 学习方式

1. **直接学习** - 使用 `/功法 学习 <功法ID>` 命令，消耗金币和灵石
2. **秘籍学习** - 使用功法秘籍学习或提升熟练度

### 学习限制

- 每个功法需要对应境界才能学习
- 功法等级不能超过角色等级
- 玩家可以同时学习多种功法，但只能装备一个

## 与技能系统的区别

| 系统 | 类型 | 效果 | 消耗 |
|------|------|------|------|
| 技能 | 主动 | 释放技能造成伤害或治疗 | 蓝量、冷却时间 |
| 功法 | 被动 | 提供属性加成和特殊效果 | 无消耗 |

技能和功法可以同时生效，共同提升角色战力。
