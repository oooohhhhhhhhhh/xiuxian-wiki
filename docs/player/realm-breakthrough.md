# 境界突破

境界突破是修仙之旅的核心成长机制。通过积累灵力、消耗灵石、渡过心魔考验来晋升到更高的修仙境界。

---

## 突破条件

### 基本条件

1. **灵力达标**：当前灵力 ≥ 当前阶段所需的突破灵力
2. **灵石充足**：拥有足够的灵石用于突破消耗
3. **通过心魔考验**：根据境界不同，会触发心魔考验

### 灵力获取

- 使用 `/修炼` 进入闭关状态，按境界倍率自动积累灵力
- 游历探索、秘境挑战、签到等也可获取灵力

---

## 突破成功率

突破成功率受以下因素综合影响：

### 基础成功率

每个境界有独立的基础突破概率。

### 灵根加成

| 灵根品级 | 渡劫成功率加成 |
|----------|--------------|
| 鸿蒙灵根 | +35% |
| 正灵根 | +30% |
| 天灵根 | +15% |
| 异灵根 | +10% |
| 双灵根 | +5% |
| 三灵根 | +2% |
| 四灵根 | +0% |
| 杂灵根 | -10%（大器晚成者境界≥4时 +15%） |

### 心魔抗性

不同灵根品级对心魔有不同的抗性修正：

| 灵根品级 | 心魔抗性修正 |
|----------|------------|
| 天灵根 | +4% |
| 异灵根 | +2% |
| 鸿蒙灵根 | 额外抗性 |
| 杂灵根 | -2% |

---

## 突破失败

突破失败时：

- 消耗的灵石**不退还**
- 灵力**保留 50%**
- 有概率触发重伤（属性暂时下降）
- 混沌灵根的大器晚成效果：突破失败时触发额外属性提升，突破成功时也有额外加成

---

## 天劫系统

突破至金丹期及以上境界时，需要渡过天劫。

### 天劫机制

- 天劫强度随境界提升而递增
- 需要足够的防御属性和生命值来承受天劫伤害
- 灵根品级越高，渡劫成功率加成越大

### 渡劫策略

1. 提升防御属性（装备、强化）
2. 利用灵根品级的渡劫加成
3. 高品级灵根（鸿蒙、正灵根、天灵根）渡劫成功率显著更高

---

## 突破后的变化

### 属性大幅提升

每次突破都会获得生命、法力、攻击、防御、速度、神识的永久加成。具体数值见[境界列表](/player/realm-list)。

### 技能解锁

每突破到新的大境界（初期），解锁一个新技能：

| 境界 | 解锁技能 | unlockSkillId |
|------|----------|:---:|
| 筑基期 | 技能1 | 1 |
| 金丹期 | 技能2 | 2 |
| 元婴期 | 技能3 | 3 |
| 化神期 | 技能4 | 4 |
| 合体期 | 技能5 | 5 |
| 大乘期 | 技能6 | 6 |
| 渡劫期 | 技能7 | 7 |
| 真仙 | 技能8 | 8 |

使用 `/学习 <技能ID或名称>` 学习新技能，具体见[技能系统](/player/skills)。

### 修炼速度提升

突破到大境界后，修炼速度倍率提升，每秒获得更多灵力。

## 各境界突破建议

### 炼气期

- 新人阶段，灵力需求低，正常修炼即可
- 使用 `/修炼` 积累灵力后 `/突破`

### 筑基期

- 灵力需求开始提升，建议配合游历获取额外灵力
- 注意灵石储备

### 金丹期

- 从此境界开始触发天劫
- 天灵根以上品级渡劫较轻松
- 解锁灵根重铸功能（`/灵根 重铸`）

### 元婴期 ~ 化神期

- 灵力需求大幅增长，修炼效率成为关键
- 灵根的修炼效率加成效果显著
- 善用灵根共鸣系统获取额外属性

### 合体期 ~ 大乘期

- 灵力以亿为单位，需要长期修炼积累
- 高品级灵根的渡劫加成至关重要
- 修炼速度倍率已达 12x ~ 20x

### 渡劫期 ~ 真仙

- 灵力以兆为单位，是漫长的修仙之路
- 突破消耗的灵石也非常惊人
- 需全方位提升角色实力

### 金仙

- 当前版本的终极境界
- 永恒不朽，万劫不灭
