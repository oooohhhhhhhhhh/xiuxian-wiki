# 装备系统

装备系统允许玩家穿戴法宝提升属性，装备可通过强化进一步提升能力。

## 装备类型

在修仙世界中，装备被称为**法宝**，属于 `TREASURE` 类型：

| 类型 | 说明 | 属性加成 |
|------|------|----------|
| TREASURE | 法宝 | 攻击、防御、速度、灵力等 |
| EQUIPMENT | 装备 | 属性加成 |

## 装备强化

### 强化规则

| 强化等级 | 成功率 | 失败后果 |
|----------|--------|----------|
| +1 ~ +5 | 95% ~ 75% | 无 |
| +6 ~ +10 | 70% ~ 40% | 降级 |
| +11 ~ +15 | 35% ~ 3% | 装备破碎 |

### 强化加成

```
强化属性加成 = 基础属性 × 强化等级 × 0.1
```

| 强化等级 | 属性加成 |
|----------|----------|
| +1 | +10% |
| +3 | +30% |
| +5 | +50% |
| +10 | +100% |
| +15 | +150% |

### 强化消耗

| 强化等级 | 强化石消耗 | 金币消耗 |
|----------|------------|----------|
| +1 | 1 | 100 |
| +3 | 3 | 300 |
| +5 | 5 | 500 |
| +10 | 10 | 1000 |
| +15 | 15 | 2000 |

## 装备管理

### QQ 指令

| 指令 | 说明 |
|------|------|
| `/装备` | 查看已装备物品 |
| `/穿戴 <物品>` | 穿戴物品 |
| `/卸下 <物品>` | 卸下物品 |
| `/强化 <物品>` | 强化装备 |

### REST API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/game/equipment/equipped` | 查看已装备 |
| POST | `/api/game/equipment/equip` | 穿戴装备 |
| POST | `/api/game/equipment/unequip` | 卸下装备 |
| POST | `/api/game/equipment/enhance` | 强化装备 |

## 装备数据一致性

### 数据修复

服务器启动时会自动执行装备数据一致性检查：

1. 移除不存在的装备（itemKey 在 ItemRegistry 中找不到的）
2. 移除背包中重复存在的装备（装备后应从背包移除）
3. 警告背包中有装备物品但未装备的情况

### 手动修复

管理员可以使用命令手动触发修复：

```
/修复装备
```

需要 `admin.fix.equipment` 权限。

## 装备搭配建议

### PVP 战斗

推荐使用高攻击法宝：

- **龙剑** + **空间戒指** + 攻击称号

### PVE 战斗

推荐使用高防御法宝：

- **凤凰甲** + **麒麟护符** + 生命称号

### 挂机修炼

推荐使用增加背包容量的法宝：

- **空间戒指** + 聚灵阵 + 经验称号
