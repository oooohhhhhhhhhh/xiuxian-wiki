## 称号管理

### QQ 指令

| 指令 | 说明 |
|------|------|
| `/称号` | 查看称号列表 |
| `/称号 装备 <称号>` | 装备称号 |
| `/称号 卸下` | 卸下称号 |

### REST API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/game/title/my` | 查看我的称号 |
| GET | `/api/game/title/active` | 查看当前装备称号 |
| POST | `/api/game/title/equip` | 装备称号 |
| POST | `/api/game/title/unequip` | 卸下称号 |

## 称号效果叠加

- 每个称号的属性加成独立计算
- 装备称号后立即获得加成
- 卸下称号后立即失去加成
- 只能装备一个称号
