# 称号系统

称号系统为玩家提供额外的属性加成，完成特定成就或达到目标后即可获得。当前版本共有 **19 个预设称号**，分为凡品、良品、珍品、仙品、神品五个等级。

> 完整的称号数据和属性加成请参考 **[称号列表](/player/title-list)**。

## 称号稀有度

| 稀有度 | 显示名称 | 说明 |
|--------|----------|------|
| COMMON | 凡品 | 基础属性加成，入门称号 |
| UNCOMMON | 良品 | 中等属性加成，需要一定条件获取 |
| RARE | 珍品 | 较高属性加成，包含百分比效果 |
| EPIC | 仙品 | 优秀属性加成，附带修炼速度等特殊效果 |
| LEGENDARY | 神品 | 顶级属性加成，终极成就 |

## 称号效果

### 固定属性加成

每个称号提供固定的属性数值加成，包括：
- **攻击 / 防御**：直接提升战斗属性
- **生命 / 法力**：增加生存能力
- **速度 / 灵力**：提升回合优先度和法术威力

### 百分比加成

部分高级称号还提供百分比效果：
- **修炼速度**：加快灵力获取
- **经验加成**：提升经验获取效率
- **掉落率**：增加物品掉落概率

## 称号管理

### 装备称号

```
/称号 装备 <称号名称>
```

每次只能装备一个称号，装备后立即获得对应属性加成。

### 卸下称号

```
/称号 卸下
```

卸下后属性加成立即移除。

### 查看称号

```
/称号
```

查看当前已获得的称号列表及装备状态。

## QQ 指令

| 指令 | 说明 |
|------|------|
| `/称号` | 查看称号列表和当前装备状态 |
| `/称号 装备 <称号名称>` | 装备指定称号 |
| `/称号 卸下` | 卸下当前称号 |

## 称号获取方式

| 获取方式 | 说明 |
|----------|------|
| 境界达成 | 达到特定境界自动获得（如炼丹学徒、首席弟子） |
| 成就解锁 | 完成特定成就获得（如副本征服者、捉虫达人） |
| 活动奖励 | 参与限时活动获得（如迎春使者 - 2026春节限定） |
| 身份达成 | 创建宗门、坊市交易等里程碑（如开山祖师、富甲一方） |

## 称号搭配建议

### PVP 战斗
推荐攻击和速度加成高的称号：**逆天者** + **剑道宗师**

### PVE 战斗
推荐生命和防御加成高的称号：**丹道尊者** + **万兽之主**

### 修炼挂机
推荐修炼速度和经验加成高的称号：**混沌帝尊** + **大道之主**
