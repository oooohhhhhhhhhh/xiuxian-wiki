# PVP对战

PVP对战是玩家之间的回合制战斗系统，支持技能施放、暴击、灵根特效等机制。

## 战斗规则

### 回合制战斗

每回合双方依次行动，行动顺序由速度决定：

```
行动顺序 = 速度值比较（速度高者先手）
速度相同 = 随机决定先后顺序
```

### 战斗流程

1. **发起挑战**：玩家 A 向玩家 B 发起挑战
2. **接受挑战**：玩家 B 在 60 秒内接受挑战
3. **战斗开始**：双方进入战斗状态
4. **回合行动**：每回合选择技能或使用物品
5. **战斗结束**：一方 HP 降至 0 或战斗超时（30回合）

### 胜负判定

| 条件 | 结果 |
|------|------|
| 对方 HP ≤ 0 | 胜利 |
| 自己 HP ≤ 0 | 失败 |
| 战斗达到 30 回合 | 平局（HP 高者胜） |

## 战斗公式

### 伤害计算

```
伤害 = (攻击 - 防御 × 0.5) × 技能倍率 × 暴击倍率 × 灵根特效倍率
```

| 因素 | 说明 |
|------|------|
| 攻击 | 角色攻击力（含 Buff 加成） |
| 防御 | 对方防御力（含 Buff 加成） |
| 技能倍率 | 技能伤害倍率（1.0 ~ 2.5） |
| 暴击倍率 | 暴击时 1.5 ~ 2.0 倍 |
| 灵根特效倍率 | 火灵根 +10%，金灵根 +5% |

### 暴击计算

```
暴击率 = 基础暴击率 + 灵根加成 + 装备加成
基础暴击率 = 5%
金灵根加成 = +5%
火灵根加成 = +3%
装备加成 = 根据装备属性
```

### 闪避计算

```
闪避率 = 基础闪避率 + 灵根加成 + 装备加成
基础闪避率 = 3%
风灵根加成 = +8%
空间灵根加成 = +10%
```

## 灵根特效

### 攻击特效

| 灵根 | 特效 | 触发条件 |
|------|------|----------|
| 金灵根 | 暴击率 +5% | 永久生效 |
| 火灵根 | 暴击伤害 +15% | 暴击时 |
| 火灵根 | 技能伤害 +10% | 使用技能时 |
| 毒灵根 | 中毒效果 | 攻击时 10% 概率 |

### 防御特效

| 灵根 | 特效 | 触发条件 |
|------|------|----------|
| 木灵根 | 每回合回复 3% HP | 每回合开始 |
| 土灵根 | 受到伤害降低 5% | 被攻击时 |
| 水灵根 | 技能蓝耗降低 15% | 使用技能时 |
| 冰灵根 | 冻结敌人 1 回合 | 被攻击时 5% 概率 |

## PVP 奖励

### 胜利奖励

| 奖励类型 | 数量 |
|----------|------|
| 经验 | 1000 + 对方等级 × 100 |
| 声望 | 20 |
| 灵石 | 100 |

### 失败惩罚

| 惩罚类型 | 数量 |
|----------|------|
| 经验 | 损失当前经验的 5% |
| 声望 | -10 |

## 战斗策略

### QQ 指令

| 指令 | 说明 |
|------|------|
| `/挑战 <玩家名>` | 向指定玩家发起挑战 |
| `/挑战 接受` | 接受待处理的切磋挑战 |
| `/挑战 拒绝` | 拒绝待处理的切磋挑战 |
| `/战斗策略` | 设置战斗策略 |

### 策略设置

玩家可以设置自动战斗策略：

| 策略类型 | 说明 |
|----------|------|
| 全力攻击 | 优先使用攻击技能 |
| 均衡战斗 | 合理分配攻击和防御 |
| 防御反击 | 先防御，后反击 |
| 消耗战术 | 消耗对方法力 |
