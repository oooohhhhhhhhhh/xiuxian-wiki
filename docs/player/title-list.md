# 称号列表

修仙世界共有 19 个预设称号，分为凡品、良品、珍品、仙品、神品五个等级。

## 称号稀有度

| 稀有度 | 显示名称 | 说明 |
|--------|----------|------|
| COMMON | 凡品 | 普通称号，基础属性加成 |
| UNCOMMON | 良品 | 稀有称号，中等属性加成 |
| RARE | 珍品 | 珍贵称号，较高属性加成 |
| EPIC | 仙品 | 史诗称号，优秀属性加成 |
| LEGENDARY | 神品 | 传说称号，顶级属性加成 |

## 称号管理

### 装备称号

```
/称号 装备 <称号名称>
```

### 卸下称号

```
/称号 卸下
```

### 查看称号列表

```
/称号
```

## 称号获取方式

| 获取方式 | 说明 |
|----------|------|
| 境界达成 | 达到特定境界自动获得 |
| 活动奖励 | 参与限时活动获得 |
| 成就解锁 | 完成特定成就获得 |
| 管理员发放 | 特殊称号由管理员发放 |

## 称号搭配建议

### PVP 战斗

推荐使用攻击和速度加成较高的称号：

- **剑道宗师** + **逆天者** + 高攻击装备

### PVE 战斗

推荐使用生命和防御加成较高的称号：

- **丹道尊者** + **万兽之主** + 高防御装备

### 修炼挂机

推荐使用经验和修炼速度加成较高的称号：

- **混沌帝尊** + **大道之主** + 聚灵阵
