# PVE战斗

PVE战斗包括游历和秘境中的妖兽战斗，以及秘境Boss战。

## 战斗触发

### 游历战斗
- **触发方式**：游历探索时随机遭遇妖兽
- **触发概率**：20%

### 秘境战斗
- **普通妖兽**：每个秘境区域都有对应的妖兽
- **Boss战**：秘境深处随机遭遇Boss（10%概率）

## 战斗技巧

### 战前准备
- 检查 HP/MP 是否充足
- 使用 Buff 物品提升属性
- 装备合适的武器和防具

### 战斗策略
- 优先攻击低防御敌人
- 注意技能冷却时间
- 合理使用回复物品
- 利用灵根特效优势
