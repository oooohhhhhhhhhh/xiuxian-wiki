# 境界突破

境界是修仙世界中衡量玩家实力的核心指标。玩家通过修炼获取经验，达到一定经验值后可以尝试突破到更高境界。

## 突破规则

### 小境界突破
- **成功率**：100%
- **消耗**：无特殊消耗
- **效果**：属性提升，修炼速度增加

### 大境界突破
大境界突破会触发**天劫**，成功率随境界递减：

| 境界 | 基础成功率 | 天劫类型 |
|------|------------|----------|
| 炼气→筑基 | 80% | 雷劫 |
| 筑基→金丹 | 70% | 心魔劫 |
| 金丹→元婴 | 60% | 风火劫 |
| 元婴→化神 | 50% | 水劫 |
| 化神→合体 | 40% | 五行劫 |
| 合体→大乘 | 30% | 阴阳劫 |
| 大乘→渡劫 | 20% | 轮回劫 |
| 渡劫→真仙 | 15% | 混沌劫 |
| 真仙→金仙 | 10% | 鸿蒙劫 |

## 突破成功率计算

突破成功率受多种因素影响：

```
最终成功率 = 基础成功率 + 灵根加成 + 灵力加成 + 丹药加成
最终成功率 = min(最终成功率, 95%)
```

### 影响因素

| 因素 | 加成效果 |
|------|----------|
| 灵根等级 | 天灵根 +10%，异灵根 +8%，双灵根 +5%，三灵根 +3%，四灵根 +2%，杂灵根 +1% |
| 灵力属性 | 灵力值 ÷ 10000 × 10%（最高 +10%） |
| 天劫对应灵根 | 拥有对应属性灵根 +5% |
| 渡劫丹 | 使用渡劫丹 +10% |

### 示例计算

假设玩家从金丹突破到元婴：
- 基础成功率：60%
- 灵根加成（异灵根）：+8%
- 灵力加成（灵力值 5000）：+5%
- 天劫对应灵根（木灵根抵抗风火劫）：+5%
- 使用渡劫丹：+10%

最终成功率 = min(60% + 8% + 5% + 5% + 10%, 95%) = 88%

## 突破失败后果

- **损失经验**：损失当前境界所需经验的 1/3
- **重伤状态**：所有属性降低 20%，持续 24 小时
- **冷却时间**：24 小时内无法再次突破

## 渡劫丹

渡劫丹是大境界突破的重要辅助道具：

| 物品 | 效果 | 获取方式 |
|------|------|----------|
| 渡劫丹 | 突破成功率 +10%，上限 95% | 制造系统合成 |
| 高级渡劫丹 | 突破成功率 +15%，上限 95% | 秘境探索获得 |
