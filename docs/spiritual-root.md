# 灵根系统

灵根是修仙世界中角色的核心属性，决定了角色的修炼方向和战斗特色。玩家创建角色时随机抽取灵根，每种灵根拥有独特的属性加成和特殊效果。

## 灵根等级

灵根分为6个等级，从高到低依次为：

| 等级 | 名称 | 说明 |
|------|------|------|
| 天级 | 天灵根 | 最稀有，属性加成最高 |
| 异级 | 异灵根 | 稀有，特殊效果最强 |
| 双级 | 双灵根 | 两种灵根组合 |
| 三级 | 三灵根 | 三种灵根组合 |
| 四级 | 四灵根 | 四种灵根组合 |
| 杂级 | 杂灵根 | 五种及以上灵根，属性加成最低 |

## 灵根类型

游戏共有15种灵根类型，每种灵根对应不同的属性加成和特殊效果：

### 金灵根
- **属性加成**：攻击 +15%，防御 +10%
- **特殊效果**：暴击率 +5%，攻击伤害 +10%

### 木灵根
- **属性加成**：生命 +20%，防御 +10%
- **特殊效果**：战斗中每回合回复 3% HP，修炼速度 +5%

### 水灵根
- **属性加成**：法力 +20%，速度 +5%
- **特殊效果**：技能蓝耗降低 15%，法力回复速度 +10%

### 火灵根
- **属性加成**：攻击 +20%，暴击伤害 +15%
- **特殊效果**：技能伤害 +10%，暴击率 +3%

### 土灵根
- **属性加成**：防御 +15%，生命 +15%
- **特殊效果**：坊市手续费减半（5%→2.5%），受到的伤害降低 5%

### 雷灵根
- **属性加成**：攻击 +18%，速度 +8%
- **特殊效果**：暴击伤害 +20%，攻击时有概率触发额外伤害

### 风灵根（巽风）
- **属性加成**：速度 +15%，闪避率 +8%
- **特殊效果**：游历冷却时间缩减 20%，战斗先手率提高

### 冰灵根
- **属性加成**：防御 +12%，速度 +5%
- **特殊效果**：攻击时有概率减速敌人，受到攻击时有概率冻结攻击者

### 毒灵根
- **属性加成**：攻击 +12%，暴击率 +5%
- **特殊效果**：攻击时有概率使敌人中毒，持续造成伤害

### 暗灵根
- **属性加成**：攻击 +15%，闪避率 +5%
- **特殊效果**：夜间战斗伤害加成，敌人难以锁定目标

### 光灵根
- **属性加成**：生命 +15%，防御 +8%
- **特殊效果**：战斗中每回合回复 2% MP，对暗属性敌人伤害加成

### 空间灵根
- **属性加成**：速度 +12%，闪避率 +10%
- **特殊效果**：瞬移能力，战斗中可瞬间调整位置

### 时间灵根
- **属性加成**：速度 +10%，攻击 +10%
- **特殊效果**：技能冷却时间缩短，修炼效率提升

### 混沌灵根
- **属性加成**：全属性 +5%
- **特殊效果**：后期觉醒特殊能力，可转化其他灵根属性

### 鸿蒙灵根
- **属性加成**：全属性 +8%
- **特殊效果**：最高级灵根，所有特殊效果增强

## 灵根抽取规则

创建角色时，系统随机抽取灵根：

1. **天灵根**：抽取概率 1%
2. **异灵根**：抽取概率 3%
3. **双灵根**：抽取概率 10%
4. **三灵根**：抽取概率 25%
5. **四灵根**：抽取概率 35%
6. **杂灵根**：抽取概率 26%

## 灵根共鸣

玩家连续活跃达到一定天数时触发灵根共鸣，永久提升属性：

| 活跃天数 | 效果 |
|----------|------|
| 7天 | 全属性 +3% |
| 30天 | 全属性 +5%，灵根效果 +10% |
