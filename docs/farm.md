# 农场系统

农场系统是玩家获取资源的重要途径。玩家拥有自己的农田，可以种植各类种子，通过浇水施肥提升产量。

## 农田规则

| 参数 | 值 |
|------|------|
| 初始地块数 | 4 块 |
| 最大地块数 | 12 块 |
| 水分上限 | 100%（每秒消耗 1%） |
| 肥力上限 | 100%（消耗肥料提升） |
| 扩建费用 | 100 × 2^(当前地块数-4) 金币 |
| 枯萎死亡时间 | 超过1小时未浇水自动死亡 |

### 扩建费用计算

```
扩建费用 = 100 × 2^(当前地块数 - 4)
```

| 当前地块数 | 扩建费用 |
|------------|----------|
| 4 → 5 | 100 金币 |
| 5 → 6 | 200 金币 |
| 6 → 7 | 400 金币 |
| 7 → 8 | 800 金币 |
| 8 → 9 | 1600 金币 |
| 9 → 10 | 3200 金币 |
| 10 → 11 | 6400 金币 |
| 11 → 12 | 12800 金币 |

## 季节系统

季节根据真实时间自动计算，影响作物生长速度：

| 季节 | 时间范围 | 加成属性 | 加成效果 |
|------|----------|----------|----------|
| 春季 | 3-5月 | 木、灵 | +25%生长速度 |
| 夏季 | 6-8月 | 火 | +30%生长速度 |
| 秋季 | 9-11月 | 金、土 | +20%生长速度 |
| 冬季 | 12-2月 | 水、冰 | +30%生长速度 |

**注意**：不匹配的属性可能会受到季节减成（如火系作物在冬季生长速度-30%）

## 病虫害系统

作物在生长过程中可能受到病虫害影响：

| 状态 | 效果 | 触发条件 |
|------|------|----------|
| 健康 | 无影响 | 默认状态 |
| 轻度虫害 | 产量-20% | 种植5分钟后随机触发 |
| 严重虫害 | 产量-40% | 轻度虫害持续30分钟后升级 |
| 病害 | 产量-30% | 种植5分钟后随机触发 |

**杀虫剂**：使用杀虫剂可清除病虫害，恢复作物健康状态。

## 种植流程

1. **选择地块**：选择一个空闲地块
2. **选择种子**：从背包中选择要种植的种子
3. **开始种植**：消耗种子，地块进入生长状态，应用灵根和季节加成
4. **浇水施肥**：保持水分和肥力在较高水平
5. **防治病虫害**：定期检查，及时使用杀虫剂
6. **等待成熟**：根据种子类型和加成等待生长时间
7. **收获作物**：成熟后收获作物，获得奖励

## 产量计算

```
实际产量 = max(1, 基础产量 + 水分加成 + 肥力加成 + 随机加成 - 病虫害惩罚)
```

| 因素 | 计算方式 |
|------|----------|
| 基础产量 | 根据种子类型，受灵根加成影响 |
| 水分加成 | 基础产量 × 水分百分比 × 0.5 |
| 肥力加成 | 基础产量 × 肥力百分比 × 肥料加成系数 |
| 随机加成 | 0~2 随机值 |
| 病虫害惩罚 | 轻度-20%，严重-40%，病害-30% |

## 枯萎恢复

- 当水分降至0%时，作物进入枯萎状态
- 枯萎后30分钟内浇水可恢复（恢复至50%水分）
- 枯萎超过1小时，作物自动死亡，必须清理后重新种植

## 种植策略

### 新手策略
- 优先种植灵草种子（生长快，产量稳定）
- 及时浇水，保持水分在 60% 以上
- 逐步扩建农田，积累资源

### 进阶策略
- 根据灵根属性选择匹配的作物，最大化加成
- 根据季节调整种植策略，避开不利季节
- 保持肥力在 80% 以上，最大化产量
- 定期检查病虫害，及时防治

### 效率最大化
- 使用一键操作提高效率
- 利用离线时间种植长周期作物
- 优先种植当前需求的材料
- 合理搭配不同生长周期的作物

### 灵根与季节配合
- 木系灵根在春季种植灵草、人参
- 火系灵根在夏季种植火焰藤
- 水系灵根在冬季种植天山雪莲
- 鸿蒙灵根可任意季节种植任意作物

## 示例计算

**种植千年人参（木系），玩家拥有青帝木灵根，当前为春季：**
- 基础产量：2
- 灵根加成：+30% → 基础产量提升至 2.6
- 季节加成（春季+木系）：+25% → 生长时间缩短
- 实际产量 = 2.6 + 水分加成 + 肥力加成 + 随机值
