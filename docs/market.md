# 坊市交易

坊市是玩家之间进行物品交易的平台，使用灵石作为交易货币。

## 交易规则

### 手续费

| 条件 | 手续费率 |
|------|----------|
| 普通玩家 | 5% |
| 土灵根玩家 | 2.5% |

```
实际收到灵石 = 挂单价格 × (1 - 手续费率)
```

### 交易限制

| 限制类型 | 值 |
|----------|------|
| 最大挂单数 | 10 单 |
| 挂单有效期 | 7 天 |
| 最小挂单价格 | 10 灵石 |
| 最大挂单价格 | 100000 灵石 |

## 交易流程

### 挂单

1. **选择物品**：从背包中选择要出售的物品
2. **设置价格**：输入出售价格（灵石）
3. **确认挂单**：系统扣除物品，创建挂单

### 购买

1. **浏览挂单**：查看坊市中的所有挂单
2. **选择商品**：选择想要购买的物品
3. **确认购买**：系统扣除灵石，将物品放入背包

### 撤单

1. **查看挂单**：查看自己的挂单列表
2. **选择撤单**：选择要撤回的挂单
3. **确认撤单**：系统返还物品，撤销挂单

## QQ 指令

| 指令 | 说明 |
|------|------|
| `/坊市` | 查看坊市挂单列表 |
| `/挂单 <物品> <数量> <价格>` | 挂单出售物品 |
| `/购买 <挂单ID>` | 购买指定挂单的物品 |
| `/撤单 <挂单ID>` | 撤回自己的挂单 |
| `/我的挂单` | 查看自己的挂单列表 |

## REST API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/game/market` | 查看挂单列表 |
| POST | `/api/game/market/list` | 挂单出售物品 |
| POST | `/api/game/market/buy` | 购买物品 |
| POST | `/api/game/market/cancel` | 撤回挂单 |
| GET | `/api/game/market/my_listings` | 查看我的挂单 |

## 交易策略

### 卖家策略
- 了解市场行情，合理定价
- 利用土灵根减少手续费
- 优先出售稀有物品获取高收益

### 买家策略
- 对比不同挂单价格
- 关注即将过期的挂单（可能有折扣）
- 批量购买可议价
