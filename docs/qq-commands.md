# QQ 机器人指令

修仙世界服务端集成了 QQ 机器人（基于 OneBot 协议），玩家可以通过 QQ 聊天与游戏交互。

## 指令格式

所有游戏指令以 `/` 开头，例如：
```
/状态
/修炼
/游历
```

部分指令支持中文别名，例如 `/状态` 和 `/info` 效果相同。

| 指令 | 别名 | 说明 |
|------|------|------|
| `/打坐` | `meditate`, `闭关`, `cultivate` | 开始挂机打坐 |

## OneBot 协议支持

机器人使用 OneBot 协议与 QQ 客户端通信，支持以下事件：

| 事件类型 | 说明 |
|----------|------|
| `message` | 接收群消息和私聊消息 |
| `friend_add` | 好友添加事件 |
| `group_member_add` | 群成员添加事件 |

---

## 注意事项

1. **私聊优先**：部分敏感指令（如 `/解绑`）仅在私聊中可用
2. **冷却时间**：探索、突破等操作有冷却时间限制
3. **权限验证**：管理员指令需要对应权限等级
4. **消息长度**：指令参数过长可能被截断，建议使用短名称
