# QQ机器人指令

修仙世界支持 OneBot 协议的 QQ 机器人，玩家可以通过 QQ 发送指令进行游戏操作。

## 通用指令

### 帮助指令

| 指令 | 说明 |
|------|------|
| `/帮助` | 查看帮助信息 |
| `/帮助 <指令>` | 查看指定指令的详细说明 |

## 指令格式说明

### 基础格式
```
/指令名 [参数1] [参数2] ...
```

### 参数说明
- 尖括号 `<>` 表示必填参数
- 方括号 `[]` 表示可选参数
- 参数之间用空格分隔

### 示例
```
/种田 种植 1 灵草种子
/pvp 张三
/挂单 灵剑 1 100
```
