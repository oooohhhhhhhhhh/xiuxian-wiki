# 秘境探索

秘境探索是玩家获取高级资源和挑战Boss的重要途径。

## 秘境列表

游戏共有8个秘境区域：

| 秘境名称 | 等级要求 | 难度 | 特殊产出 |
|----------|----------|------|----------|
| 迷雾森林 | 炼气 | 低 | 灵草、低级材料 |
| 冰原秘境 | 筑基 | 低-中 | 冰系材料、暗冰草 |
| 火山秘境 | 筑基 | 中 | 火系材料、火焰藤 |
| 幽冥秘境 | 金丹 | 中 | 幽冥花、稀有材料 |
| 雷霆秘境 | 元婴 | 中-高 | 雷系材料、技能书 |
| 混沌秘境 | 化神 | 高 | 混沌材料、高级装备 |
| 鸿蒙秘境 | 合体 | 极高 | 鸿蒙材料、传说装备 |
| 仙界秘境 | 渡劫 | 终极 | 仙级材料、神器 |

## 秘境事件

每个秘境中随机遭遇以下事件：

| 事件类型 | 概率 | 说明 |
|----------|------|------|
| 宝藏 | 20% | 获得灵石和物品 |
| 妖兽 | 30% | 进入战斗 |
| Boss | 10% | 挑战Boss |
| 灵草 | 15% | 采集灵草 |
| 陷阱 | 15% | 受到伤害 |
| 古修 | 5% | 获得功法或经验 |
| 遗迹 | 5% | 获得稀有装备 |

## 探索策略

### 战前准备
- 检查境界是否达到要求
- 准备充足的回复物品
- 使用Buff提升属性

### 探索技巧
- 优先探索低难度秘境积累资源
- 组队探索提高成功率
- 利用灵根特效克制Boss属性
