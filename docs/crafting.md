# 制造系统

制造系统包括炼丹和炼器，玩家可以通过配方合成各种物品。

## 制造规则

### 成功率计算

```
成功率 = 基础成功率 + 熟练度加成 + 材料品质加成
失败安慰 = 获得少量制造经验
```

| 因素 | 加成效果 |
|------|----------|
| 基础成功率 | 根据配方（70% ~ 95%） |
| 熟练度加成 | 每级 +2%（最高 +20%） |
| 材料品质加成 | 使用高品质材料 +5% |

### 熟练度

- 使用制造技能获得熟练度
- 熟练度达到一定值升级
- 等级越高，成功率越高

## 制造策略

### 新手策略
- 优先制造回血丹和回蓝丹
- 积累制造经验提升熟练度
- 逐步尝试高级配方

### 进阶策略
- 批量制造高需求物品
- 利用熟练度加成提高成功率
- 关注市场需求，制造畅销物品
