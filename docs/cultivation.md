# 修炼系统

修炼是修仙世界中角色成长的核心方式。玩家通过挂机修炼获取经验，经验达到一定值后可以突破境界。

## 修炼机制

### 修炼模式
- **主动修炼**：玩家手动点击修炼按钮开始
- **自动修炼**：玩家离线后自动进入修炼状态（效率 50%）

### 修炼速度计算

```
修炼速度 = 基础速度 × 境界倍率 × 灵根倍率 × 心法加成 × 天象加成
```

| 因素 | 说明 |
|------|------|
| 基础速度 | 10 经验/秒 |
| 境界倍率 | 随境界提升，每大境界 +20% |
| 灵根倍率 | 木灵根 +5%，混沌灵根 +10% |
| 心法加成 | 装备修炼类心法，每级 +12% |
| 天象加成 | 紫气东来 +50%，星辰耀天 +30% |

## 心魔系统

修炼过程中可能触发心魔，心魔有概率导致修为倒退：

### 心魔类型

| 类型 | 概率 | 后果 |
|------|------|------|
| 轻微心魔 | 30% | 损失当前经验的 5% |
| 中等心魔 | 15% | 损失当前经验的 10% |
| 严重心魔 | 5% | 损失当前经验的 20%，境界倒退一期 |

### 心魔抵抗

心魔抵抗能力与灵力属性相关：

```
心魔抵抗率 = 灵力值 ÷ 10000 × 50%（最高 50%）
```

拥有水灵根的玩家心魔抵抗率额外 +10%。

## 晨修·紫气东来

每日一次的修炼收益加成活动：

| 连续天数 | 加成效果 |
|----------|----------|
| 1天 | 修炼速度 +20%，持续 1 小时 |
| 3天 | 修炼速度 +30%，持续 1 小时 |
| 7天 | 修炼速度 +50%，持续 1 小时 |
| 14天 | 修炼速度 +80%，持续 1 小时 |
| 30天 | 修炼速度 +100%，持续 2 小时 |

## 离线收益

玩家断线后修炼继续进行：

| 项目 | 规则 |
|------|------|
| 修炼效率 | 50% |
| 上限时长 | 8 小时 |
| 经验结算 | 上线时自动结算 |
| HP/MP恢复 | 上线时按离线时间恢复 |
| 心魔判定 | 降频（在线的 50%） |

## 今日机缘

嵌入游戏行为的每日目标系统：

### 每日目标

| 目标 | 奖励 |
|------|------|
| 游历 3 次 | 经验 +1000，灵石 +50 |
| PvP 对战 1 次 | 经验 +500，声望 +10 |
| 秘境探索 1 次 | 经验 +800，灵石 +30 |
| 学习技能 1 次 | 经验 +300，熟练度 +10 |

### 奖励发放

完成目标后当场发放奖励，无需手动领取。
